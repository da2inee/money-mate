# MoneyMate 백엔드로 자바/스프링 공부하기

이 프로젝트는 **Spring Boot + MyBatis** 구조라서, 백엔드 자바 공부용으로 보기 좋습니다.

---

## 1. 전체 구조 (레이어)

```
[클라이언트(React)]  →  [Controller]  →  [Service]  →  [Mapper]  →  [DB]
```

- **Controller**: HTTP 요청 받고, 응답 반환 (API 진입점)
- **Service**: 비즈니스 로직 (여기서는 대부분 DB 호출만)
- **Mapper**: MyBatis 인터페이스 → SQL 실행
- **Model**: DB 테이블/JSON과 맞는 자바 클래스

---

## 2. 공부 순서 추천

### ① 진입점: `MoneyMateServerApplication.java`
- `@SpringBootApplication`: 스프링 부트 앱 시작
- `main()`에서 `SpringApplication.run()` 호출

### ② API가 어떻게 열리는지: Controller
- **ExpenseController.java** (지출 API)
  - `@RestController`, `@RequestMapping("/expenses")` → `/expenses`로 오는 요청 처리
  - `@GetMapping` → GET (조회)
  - `@PostMapping` → POST (등록)
  - `@DeleteMapping("/{id}")` → DELETE (삭제)
  - `@RequestParam`, `@RequestBody`, `@PathVariable` 차이 보기
- **BudgetController.java** (예산 API)
- **TravelerController.java** (같이 간 사람 API)

### ③ 로직이 있는 곳: Service
- **ExpenseService.java**
  - Controller가 부르는 메서드들
  - 실제 일은 Mapper에 넘김
- **BudgetService.java**, **TravelerService.java**도 같은 패턴

### ④ DB 접근: Mapper + XML
- **ExpenseMapper.java** (인터페이스)
  - 메서드 이름 = XML의 SQL id와 연결
- **src/main/resources/mybatis/mapper/ExpenseMapper.xml**
  - 실제 SQL (SELECT, INSERT, DELETE 등)
  - `#{}` 로 파라미터 바인딩

### ⑤ 데이터 형태: Model / DTO
- **Model**: DB/API 공통 (예: `Expense.java`, `Budget.java`)
- **dto 패키지**: 요청용 (예: `BudgetUpdateRequest`, `TravelerRequest`)

---

## 3. 한 흐름 따라가기 (예: 지출 추가)

1. **프론트** → `POST /expenses` + JSON body
2. **ExpenseController** `createExpense(@RequestBody Expense expense)` 로 받음
3. **ExpenseService** `saveExpense(expense)` 호출
4. **ExpenseMapper** `insertExpense(expense)` → MyBatis가 XML의 INSERT 실행
5. DB에 저장 후, Controller가 `ResponseEntity.ok(savedExpense)` 로 응답

이 한 번만 따라가 보면 Controller → Service → Mapper → DB 흐름이 보입니다.

---

## 4. 키워드로 검색해 보면 좋은 것

- `@Autowired` → 의존성 주입 (스프링이 객체 넣어 줌)
- `@RestController` vs `@Controller`
- `ResponseEntity` → HTTP 상태 코드 + body 지정
- MyBatis `@Mapper` / XML `namespace`, `id`
- `@RequestParam` vs `@RequestBody` vs `@PathVariable`

---

## 5. 서버 실행해서 확인

```bash
# JDK 17 필요
cd money-mate-server
./gradlew bootRun
```

- 기본 포트: **8080**
- 예: 지출 목록 조회 → `GET http://localhost:8080/expenses?category=jeju`
